# mydustchat

#### 注意事項:
- 為了防止開分帳，所以用使用者IP再經過一些處理當作聊天ID
- 傳圖片網址可以在聊天室內顯示圖片(有些沒有圖片副檔名的網址將無法辨識)
- 每次傳訊息文字上限為300字
- 訊息上限為100則，在存檔時如果超過將會從最早的訊息開始刪除
- 伺服器30分鐘內沒人會自動關閉，再開約需等20秒
- 伺服器每次重啟都會改變ID的產生方式
- 聊天室伺服器在美國，資料庫在台灣，所以可能連線會有點延遲
- 聊天室內使用者傳什麼我不負責任喔
